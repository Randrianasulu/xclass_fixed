Introduction

The rx320 application is a Unix control program to use with the Ten-Tec RX320 shortwave receiver.

Features

  • Easy to use graphical user interface.
  • Communication with the receiver using any available serial port.
  • Virtually free from the "rubber-band" effect produced by the low serial transfer rate of the RX320
  • All RX320 features supported.
  • Dual VFO.
  • Passband tuning.
  • S-meter with signal peak indicator.
  • Band scan.
  • Frequency database file management, in Clifton Turner's *.320 format or Ten-Tec's *.cdf export format.
  • On-line help.
