Direct Client Connection (DCC)

  • Chat
  • File
DCC chat messages appear on a separate window, which can be used to have a separate, secure conversation.

All standard IRC text attributes (bold, underline and reverse video) are supported, as well as mIRC color extensions and ANSI color codes.

DCC chat windows recognize only one ircII-style command:

/me action Sends an action text to the channel. Used to tell the other user about what you are doing.
