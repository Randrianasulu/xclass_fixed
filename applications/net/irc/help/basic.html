Basic Operation

  • Connecting to a server
  • Disconnecting from the server
  • Changing your current nick name
  • Joining a channel
  • Leaving the channel

To connect to a server

  1. On the IRC menu, click Connect to.

  2. In the dialog box that appears, enter the hostname, port, set the nick and/or irc name, then click Connect.
Note
  • You can also setup a list of your preferred servers.

To disconnect from the server

  • On the IRC menu, click Disconnect.

To change your current nick name

  • On the IRC menu, go to the Servers submenu, then click Nick.
Note:
  • If you have a channel window already open, you can use the /nick command to change your nick.

To join a channel

  • Type the channel name in the Join entry box below the toolbar (do not forget the # symbol), and press Enter.
Note
  • If the channel requires a key, type it after the channel name, separated by a space, for example: #vipchannel secretkey.

To leave a channel

  • In the channel window, click on the Channel menu, then select Leave.
Note
  • You can also type the /leave command on the channel input text box to leave the channel.
