Private Chat Windows

Private messages appear on a separate window, which look similar to the channel windows, but without the nick buttons and the topic text box. All standard IRC text attributes (bold, underline and reverse video), mIRC and ANSI color codes are supported here as well.

Private chat windows recognize the following ircII-style commands:

/me action Sends an action description.
/notice message Sends a notice to the other party.
/msg nick Open a new window for a private chat with nick.
/raw command Send the specified command to the server as-is.
/quote command Same as /raw.
/whois nick Shows information about the specified nick.
/whowas nick Shows information about who used the given nick last.
