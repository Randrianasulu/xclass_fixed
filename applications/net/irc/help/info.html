The Info Window

This is the window that apperas when the program is started. Connection messages, server notices, the MOTD and some other miscellaneous information are shown in this window.

This is fOXIrc's main window. When you close it, the connection to the server is terminated, all the other currently open windows are closed and the application exits.

The following features are not yet implemented:

  • Ignores
  • Channel monitoring
  • Channel encryption (CTCP SED)
  • Auto-log
  • DCC resume
